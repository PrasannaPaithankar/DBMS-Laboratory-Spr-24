# DBMS-Laboratory-Spr-24

### Local setup
```bash
git clone https://github.com/PrasannaPaithankar/DBMS-Laboratory-Spr-24.git
cd DBMS-Laboratory-Spr-24
sudo python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
deactivate
```
### Run
```bash
source venv/bin/activate
flask run
```
